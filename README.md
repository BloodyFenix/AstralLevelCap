# Astral Level Cap

Мод для Minecraft 1.12.2, который увеличивает максимальный уровень перков в Astral Sorcery с 30 до 100 (настраивается).

## Описание

По умолчанию в Astral Sorcery максимальный уровень перков ограничен 30-м уровнем. Этот мод использует ASM трансформацию для изменения этого ограничения.

## Особенности

- Увеличивает максимальный уровень с 30 до 100 (по умолчанию)
- Настраиваемый максимальный уровень через конфигурационный файл
- Диапазон настройки: от 30 до 200 уровней
- Совместим с SevTech Ages

## Установка

1. Скомпилируйте мод с помощью Gradle:
   ```bash
   ./gradlew build
   ```

2. Скопируйте файл `build/libs/astral-level-cap-1.0.0.jar` в папку `mods` вашего сервера/клиента

3. Запустите игру/сервер

## Конфигурация

После первого запуска в папке `config` появится файл `astral-level-cap.cfg`, где можно настроить максимальный уровень:

```
max_level=100
```

Допустимые значения: от 30 до 200

## Требования

- Minecraft 1.12.2
- Minecraft Forge 14.23.5.2854 или выше
- Astral Sorcery 1.10.25 или совместимая версия

## Как это работает

Мод использует ASM (bytecode manipulation) для изменения константы максимального уровня в классе `PerkLevelManager` мода Astral Sorcery. Трансформер находит инструкцию `BIPUSH 30` в методе `ensureLevels` и заменяет её на настроенное значение.
